### Setup and Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/NizamAashiqIsmail/dashboard.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open in browser:

   ```
   http://localhost:3000
   ```
